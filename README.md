<h2>Overview</h2>

<p>
TeamDebugger is a project developed during the <b>CodeRush 1.0 Hackathon</b>.
The goal of the project was to build a functional web application within the
limited timeframe of the competition while collaborating as a team.
</p>

<p>
The platform integrates several healthcare-oriented tools such as symptom
assistance, chatbot interaction, image-based analysis interfaces, and
user authentication features.
</p>

<hr>

<h2>Key Features</h2>

<ul>
<li>User login and authentication interface</li>
<li>Interactive chatbot support</li>
<li>Disease AI interface</li>
<li>Medical report interface</li>
<li>Image scanning interface</li>
<li>Multiple side utility features integrated into the system</li>
<li>Structured homepage for navigation between features</li>
</ul>
